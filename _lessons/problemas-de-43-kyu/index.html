Semeai (ataque mútuo)

{% problem 43K-1-semeai %}

é a vez das pretas. Dois grupos se cercando.
Ou as 3 pretas ou as 3 brancas irão triunfar.

{% endproblem %}

Como acima, quando brancas e pretas se cercam e nenhum dos grupos têm 2 olhos, ambos os lados tentam preencher os espaços vagos, ou liberdades, para matar as pedras do adversário.
Essa situação é chamada de "Semeai" - ataque mútuo.

No porblema acima, ambos os lados têm 3 liberdades. Entretanto, as pretas podem matar as brancas uma jogada antes pois era a vez das pretas jogarem.
Se fosse a vez das brancas, elas ganhariam o Semeai.

{% problem 43K-2-semeai %}

As pretas podem vencer o Semeai.

{% endproblem %} {% problem 43K-3-semeai %}

Salve as três pedras pretas deixadas sozinhas no canto!

{% endproblem %}
